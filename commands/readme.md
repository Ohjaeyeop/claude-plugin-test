# README 생성 커맨드

현재 프로젝트 구조를 분석하여 README.md 파일을 생성합니다.

## 지침

1. 먼저 파일 검색 도구를 사용하여 프로젝트 구조를 파악하세요:

   - 프로젝트 타입 (Node.js, Python, Go 등)
   - 주요 소스 파일 및 디렉토리
   - 설정 파일 (package.json, pyproject.toml, go.mod 등)
   - 기존 문서

2. 핵심 파일을 읽어 다음을 파악하세요:

   - 프로젝트 이름과 설명
   - 의존성 및 요구사항
   - 진입점과 주요 기능
   - 빌드/실행 스크립트

3. 다음 섹션을 포함하는 README.md 파일을 생성하세요:

   - 프로젝트 제목 및 설명
   - 주요 기능 (파악 가능한 경우)
   - 설치 방법
   - 사용 예시
   - 프로젝트 구조 개요
   - 라이선스 (발견된 경우)

4. 프로젝트 루트 디렉토리에 README.md 파일을 작성하세요.

## 출력 형식

README 문서화 모범 사례를 따르는 잘 구조화된 마크다운 파일을 생성하세요. 명확한 제목, 명령어용 코드 블록, 간결한 설명을 사용하세요.
